Dataset Retrieval Methodology

Based on University of Michigan "Missing Government Websites and Data" guide

Layered Recovery Stack:
  • Harvard LIL + Source Cooperative - 16 TB Data.gov mirror (311k datasets, 2024‚Äì2025)
  • Find Lost Data (UMich) - Federated search across archives
  • DataLumos (ICPSR) - DOI-assigned persistent datasets
  • Internet Archive Wayback - Historical file captures
  • EDGI Climate & Justice - Environmental/justice datasets
  • End of Term Archive - Presidential transition captures
  • Webrecorder US Gov - High-fidelity website archives
  • UCSB Public Data Git - NASA/NOAA/DOE mirrors
Recovery Workflow:
  1. Identify dataset metadata (title, agency, Data.gov ID, landing URL)
  2. Search sources in priority order (1-8)
  3. Stop early if high-confidence result found (>80%)
  4. Generate provenance chain and status badges
  5. Export results in multiple formats
